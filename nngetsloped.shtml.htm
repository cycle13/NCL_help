nngetsloped

Retrieves a slope at a specified coordinate position.

Prototype

	function nngetsloped (
		i [1] : integer,  
		j [1] : integer   
	)

	return_val [1] :  double

Arguments

i

A subscript indexing the first dimensioned variable in the output grid of the most recent call to natgrid or natgridd.

j

A subscript indexing the second dimensioned variable in the output grid of the most recent call to natgrid or natgridd.

Return value

The slope (a scalar value) at the specified coordinate position (if slopes were computed during the last call to natgrid or natgridd).

Description

This function is part of the Natgrid package, which implements a natural neighbor interpolation method. Much useful information is available at the above link, including the descriptions of many control parameters that can be modified to materially change its behavior. (The functions nngetp and nnsetp are used to access these parameters.)

nngetsloped is used to retrieve values which may have been computed during the last call to natgrid or natgridd. Note that these values will only have been computed if you set the value of the control parameter "sdi" appropriately.

See Also

nngetslope

    end do
